AWS Switch Role

Simplifying Access and Enhancing Security

AWS Switch Role simplifies user access to different AWS accounts, enabling users to assume roles across accounts seamlessly. It streamlines user management and access control, providing a centralized approach. Users can switch roles securely, and administrators get a unified dashboard for efficient access management. Integration with identity providers, like Active Directory, enhances authentication simplicity and security.

Step 1: Log in to Account-B

Log in to the AWS Management Console using your credentials for Account-B.

Step 2: Navigate to IAM Roles in Account-B

Navigate to the IAM service and select "Roles" in Account-B.

Step 3: Create a Role in Account-B

A. Click on "Create role."

B. Choose "Another AWS account" as the type of trusted entity.

C. Select Another AWS Account

D. Enter the Account-A AWS account ID in the Account ID field.

D. Click on"Next"

Step 4: Attach a Policy and Create a Role in Account-B

A. Attach a policy, for example, AdministratorAccess.

B. Complete the role creation process.

C. Name the Role

D. Create Role

Step 5: Log in to Account-A

Log in to the AWS Management Console using your credentials for Account-A.

Step 6: Create a User in Account-A

A. Navigate to IAM and select "Users."

B. Click on "Create User"

C. Provide a username and click "Next"

D. Do not add any permissions to the user.

E. Click "Create user."

Step 7: Create an In-Line Policy for IAM User in Account-A

A. After creating the user, go to the user details.

B. Click on "Add Permissions" tab.

C. Click "Create in-line policy"."

D. Select "JSON".

E. While creating the in-line policy, add the Account-B IAM role ARN in the policy.

F. Click "Next"

Step 8: Review and Create

A. Name the Policy

B. Review your settings and create the IAM user with the attached in-line policy.

Step 9: Switch Role in Account-A

A. Log in to the AWS Management Console using your IAM user credentials in Account-A.

C. Click "Switch Role."

D. Enter the Account-B AWS account ID, the role name you created in Account-B, a display name of your choice and Select a "Colour".

Step 10: Click Switch Role

Click "Switch Role" to assume the role in Account-B.

Congratulations!
